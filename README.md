# Developer Portfolio Frontend

A modern, responsive personal portfolio built with React and Tailwind CSS.

## Features

* Responsive design
* Modern UI
* About section
* Skills section
* Projects section
* Contact form
* Admin Login
* Protected Admin Dashboard
* Dynamic data from REST API

## Tech Stack

* React.js
* React Router
* Tailwind CSS
* Axios
* React Icons
* Type Animation

## Backend

The frontend consumes data from the Portfolio Backend API.
